# 🚀 AI Resume Screening System

## 📌 Overview

This project is an AI-powered tool that helps recruiters automatically screen and rank resumes based on a job description.

## ⚙️ Features

- Upload multiple resumes (PDF)
- AI-based candidate evaluation
- Match score (0–100)
- Strengths & gaps analysis
- Final recommendation
- Candidate ranking
- CSV export

## 🧠 Tech Stack

- Python
- Streamlit
- Gemini API
- PyPDF2
- Pandas

## ▶️ How to Run

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

2. Open the app and provide your Gemini API key in the sidebar configuration.

3. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```

## 📊 Sample Output

- Candidate ranking table
- Top candidate highlighted
- Downloadable CSV report
